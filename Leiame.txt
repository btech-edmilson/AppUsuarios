1. Tenho que criar uma conta na Play Store?
Sim, você precisa criar uma conta de desenvolvedor no Google Play Console (não é a conta comum da Play Store para usuários).

Passos rápidos (2025):

Acesse play.google.com/console.
Faça login com sua conta Google (Gmail).
Escolha tipo: Personal (para indivíduos) ou Organization (para empresas).
Forneça dados: Nome, endereço, telefone, e-mail. Inclua verificação de identidade (documento como RG/CPF ou passaporte; para organizations, documentos da empresa).
Nova exigência (desde 2024/2025): Verifique acesso a um dispositivo Android via app Play Console (baixe o app e confirme).
Ative autenticação de 2 fatores (obrigatório).


Tempo: 5-10 minutos para criar, mas verificação pode levar 1-3 dias (Google revisa documentos).
Dica: Use uma conta Google dedicada para desenvolvedor.

4. Para eu alterar o ícone, necessito de gerar novamente o APK?
Sim, mas é simples e rápido (5 minutos + build). No Expo EAS:

Passos:

Coloque o novo ícone (PNG 1024x1024) em ./assets/icon.png.
No app.json, confirme: "icon": "./assets/icon.png".
Rode npx expo prebuild --clean (gera arquivos nativos atualizados).
Gere novo build: npx eas build --platform android --profile preview.


O EAS regenera ícones otimizados para todos tamanhos (mdpi, hdpi, etc.).


Tempo: Build leva 5-10 min, mas você baixa o novo APK via QR Code.
Dica: Para iOS, use o mesmo processo. Teste com npx expo start -c antes do build.
