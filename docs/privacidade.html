AppUsuarios — Política de Privacidade

Última atualização: 06 de março de 2026

Esta política descreve como o aplicativo AppUsuarios, desenvolvido pela BTech — Serviços em Informática, coleta, utiliza e protege as informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos descritos neste documento.

1. Sobre o Aplicativo

O AppUsuarios é um aplicativo mobile desenvolvido para gerenciamento interno de clientes e usuários. Ele permite cadastrar, editar, consultar e excluir registros de clientes por meio de uma interface simples e intuitiva.

2. Dados Coletados

O aplicativo coleta e armazena os seguintes dados dos clientes cadastrados:

  • Campo Nome completo
  • Campo Número de celular
  • Campo CEP
  • Campo Bairro
  • Campo Valor
  • Campo Status (Ativo / Inativo)
  • Campo Data de cadastro e alteração

3. Finalidade dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Gerenciamento interno de clientes pela BTech
  • Consulta, edição e controle de status dos clientes
  • Registros administrativos e operacionais

4. Armazenamento dos Dados

Os dados são armazenados em servidor próprio da BTech (esbsouza.com.br). Não utilizamos serviços de terceiros para armazenamento de dados de clientes.

5. Compartilhamento de Dados

✔ Garantia Os dados coletados não são compartilhados, vendidos ou transferidos para terceiros sob nenhuma circunstância.

6. Segurança

Adotamos medidas técnicas para proteger os dados contra acessos não autorizados, alterações indevidas ou divulgação. O acesso ao aplicativo é restrito aos usuários autorizados pela BTech.

7. Direitos do Titular dos Dados

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), o titular dos dados tem direito a:

  • Solicitar acesso aos seus dados cadastrados
  • Solicitar correção de dados incorretos
  • Solicitar a exclusão de seus dados

Para exercer esses direitos, entre em contato pelo e-mail abaixo.

8. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Recomendamos que você a revise regularmente. A data da última atualização está sempre indicada no topo deste documento.
